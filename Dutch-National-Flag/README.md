# Problem Description

#### Dutch National Flag

You are given an array with only 0s,1s and 2s.

### Constraints

    - Do in O(n)
    - Do not count the number of elements.

### Input Format

1st line :- Number of elements in the array
2nd line :- 0s , 1s and 2s as input by space seperated

### Output Format

Sorted Array
